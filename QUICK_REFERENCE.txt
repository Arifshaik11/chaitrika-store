┌──────────────────────────────────────────────────────────────────────────┐
│ 🎯 HOW IT WORKS                                                          │
└──────────────────────────────────────────────────────────────────────────┘

ADMIN ADDS PRODUCT WITH IMAGE:

  Step 1: Admin fills form in Admin Panel
  ├─ Name: "Magnetic Photo Frame"
  ├─ Image: Uploads photo.jpg
  ├─ Sizes: [4x6 @₹299, 5x7 @₹399]
  └─ Shapes: [Square, Round]

  Step 2: Image uploads to Firebase Cloud Storage
  ├─ Location: chaitrika-store.firebasestorage.app
  ├─ Path: /products/1234567890.jpg
  └─ URL: https://storage.googleapis.com/.../products/1234567890.jpg

  Step 3: Product data saved to Firestore Database
  ├─ Database: chaitrika-store
  ├─ Collection: products
  └─ Fields: name, image, sizes, shapes, price, etc.

  Step 4: Website listens for changes (Real-time Sync)
  ├─ ProductContext has onSnapshot() listener
  ├─ Detects: New product in Firestore
  ├─ Updates: React state with new product
  └─ Triggers: Re-render of all components

  Step 5: Homepage displays new product
  ├─ ProductCard shows: Product image, name, price
  ├─ Image loads: From Firebase Cloud Storage URL
  ├─ No refresh: Page updates automatically!
  └─ Result: ✨ Product visible to customers!
