================================================================================
  HOW IT WORKS
================================================================================

ADMIN WORKFLOW:
  Admin logs in → Adds product with image → Image uploads to Firebase Storage
  → Product data goes to Firestore → Website listens to Firestore → Product
  appears on homepage automatically WITHOUT refresh

CUSTOMER WORKFLOW:
  Customer browses → Selects product → Customizes with photo → Adds to cart
  → Checkout → Sends WhatsApp to admin

DATA STORAGE:
  Images: Firebase Cloud Storage (chaitrika-store.firebasestorage.app)
  Product Data: Firestore Database (chaitrika-store)
  Shopping Cart: Local Browser Storage (no server needed)

REAL-TIME SYNC:
  ProductContext uses onSnapshot() listener
  When admin adds/edits/deletes product, all components using useProducts()
  automatically receive updates within 1 second
