✅ FEATURE: Create Custom Product Categories

Now you can add UNLIMITED custom categories directly from the Admin Panel!

Before: Fixed 4 categories (Magnetic, Keychain, Acrylic, MDF)
After:  Fixed 4 + UNLIMITED custom categories


════════════════════════════════════════════════════════════════════════════════
  HOW TO USE
════════════════════════════════════════════════════════════════════════════════

1. Go to:  http://localhost:3000/admin
2. Login:  admin / admin123
3. Click:  "Add Product" button
4. In the form, next to Category dropdown:
   → See green button: "+ Add Category"
5. Click:  "+ Add Category"
6. Type:   Your category name (e.g., "Wedding", "Premium", "Gifts")
7. Click:  "Add" button
8. Result: Category added to dropdown instantly!
9. Select: Your new category
10. Fill:  Rest of product form and add product
