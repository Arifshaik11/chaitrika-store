## 🎯 **HOW TO USE**

### **Adding New Products:**
1. Go to `yoursite.com/admin`
2. Login with admin credentials
3. Click "Add Product" button
4. Fill in product details:
   - Product name
   - Category (or create new category)
   - Upload image
   - Description
   - Add sizes with prices
   - Add shapes with prices
5. Click "Add Product"
6. **Product appears on live site within 1-2 seconds!**

### **Managing Existing Products:**
- ✏️ **Edit:** Click pencil icon in admin panel
- 🗑️ **Delete:** Click trash icon (asks for confirmation)
- 👀 **View:** Check products list in admin panel
